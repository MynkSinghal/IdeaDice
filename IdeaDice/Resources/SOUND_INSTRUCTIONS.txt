== HOW TO ADD SOUND TO IDEA DICE ==

1. Download a dice roll sound effect (MP3 format) from:
   - Free Sound: https://freesound.org/search/?q=dice+roll
   - ZapSplat: https://www.zapsplat.com/sound-effect-category/dice/
   - Pixabay: https://pixabay.com/sound-effects/search/dice/

2. Rename the downloaded file to "dice_roll.mp3"

3. In Xcode:
   - Select your IdeaDice project in the Navigator panel
   - Click "File" > "Add Files to IdeaDice..."
   - Navigate to and select your dice_roll.mp3 file
   - Make sure "Copy items if needed" is checked
   - In the "Add to targets" section, ensure "IdeaDice" is checked
   - Click "Add"

4. Verify sound is in your bundle:
   - The file should appear in your project navigator
   - Check that it's included in "Target Membership" (visible in the File Inspector)
   - Build and run to test
